💬 Basic Python Chat Box

📜 Project Overview

This project is a simple command-line chatbot built using Python. It interacts with users by responding to basic greetings, common questions, and joke requests. The chatbot uses predefined responses and selects replies randomly to make conversations feel less repetitive.

The project demonstrates fundamental Python concepts and provides a beginner-friendly introduction to building interactive programs.

🧠 Key Python Concepts Used

Dictionaries for storing chatbot responses
Functions for organizing code (chat_data, main)
Loops (while True) for continuous interaction
Conditional statements (if-elif-else)
Random module (random.choice) for varied responses
User input/output (input, print)

⚙️ How It Works

The chatbot starts by displaying a welcome message.
The user is given two options:
Ask a question
Exit
If the user chooses to ask a question:
The chatbot takes input and converts it to lowercase.
It checks if the input matches any predefined question.
If a match is found, a random response is displayed.
If not, it returns a default message.
If the user chooses to exit, the program ends.

▶️ How to Run

Make sure Python is installed on your system.
Save the file as Basic_Chat_Box.py.
Open terminal or command prompt.

Run the program using:

python Basic_Chat_Box.py
